Resumo: Flexbox em CSS

O que é Flexbox?

Flexbox (ou Flexible Box Layout) é um modelo de layout do CSS3 que facilita a organização de elementos em uma página HTML de forma dinâmica e flexível. Ele permite que os itens dentro de um container se ajustem automaticamente, independentemente de seu tamanho ou da resolução da tela.

Como funciona?

O Flexbox funciona com base em dois tipos de elementos:

  • Container Flexível: o elemento pai que recebe as regras de layout;
  • Itens Flexíveis: os elementos filhos, que serão organizados conforme as regras definidas no container.

Propriedades do Container

  • display: flex; – ativa o modo Flexbox no container.
  • flex-direction: define a direção dos itens (row, column, row-reverse, column-reverse).
  • flex-wrap: define se os itens devem quebrar linha (nowrap, wrap, wrap-reverse).
  • flex-flow: forma abreviada de flex-direction e flex-wrap.
  • justify-content: alinha os itens no eixo principal (start, center, space-between, etc).
  • align-items: alinha os itens no eixo secundário (stretch, center, baseline, etc).
  • align-content: alinha as linhas do conteúdo quando há múltiplas linhas.

Propriedades dos Itens

  • order: define a ordem de exibição dos itens (valores numéricos, padrão 0).
  • flex-grow: define a capacidade do item de crescer para ocupar o espaço disponível.
  • flex-shrink: define a capacidade do item de encolher se necessário.
  • flex-basis: define o tamanho base do item antes da distribuição do espaço.
  • flex: atalho para flex-grow, flex-shrink e flex-basis.
  • align-self: sobrescreve o align-items para um item específico.

Como usar Flexbox para criar layouts

Flexbox é extremamente útil para construir layouts modernos e responsivos. Abaixo estão os passos essenciais para criar um layout usando Flexbox.

1. Defina o container flexível


    .container {
    display: flex;
    }

Isso ativa o Flexbox no elemento container. Todos os filhos diretos se tornam "flex items".

2. Escolha a direção dos itens

Use flex-direction para definir se os itens ficam em linha ou em coluna:


    .container {
    flex-direction: row;        /* em linha (padrão) */
    /* flex-direction: column; */  /* empilhado verticalmente */
    }

3. Permitir quebra de linha (opcional)


    .container {
    flex-wrap: wrap;
    }

Isso permite que os itens "quebrem linha" caso não caibam no espaço disponível.

4. Distribua o espaço entre os itens

Use justify-content para alinhar os itens no eixo principal:


    .container {
    justify-content: space-between; /* espaço entre os itens */
    /* Outros valores: center, flex-start, flex-end, space-around, space-evenly */
    }

5. Alinhe os itens no eixo cruzado (vertical por padrão)


    .container {
    align-items: center;
    }

Alinha os itens verticalmente (ou horizontalmente, se em coluna).

6. Distribua múltiplas linhas (quando wrap estiver ativado)


    .container {
    align-content: space-around;
    }

7. Controle o crescimento dos itens


    .item {
    flex-grow: 1;
    }

Todos os itens com flex-grow: 1 vão crescer igualmente para ocupar o espaço disponível.

8. Use o atalho flex para simplificar


    .item {
    flex: 1; /* equivale a flex-grow: 1; flex-shrink: 1; flex-basis: 0% */
    }

9. Exemplo prático de layout com 3 colunas


    .container {
    display: flex;
    gap: 10px;
    }
    .item {
    flex: 1;
    background: #ddd;
    padding: 20px;
    }

Esse código cria três colunas que se ajustam automaticamente à largura da tela.
