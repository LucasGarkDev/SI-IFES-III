✅ Etapa 1: Análise Visual do Layout

Antes de escrever qualquer HTML ou CSS, observe o layout como um todo, como se fosse um quebra-cabeça.

Você precisa identificar:

🧱 Blocos principais

Quais partes estão claramente separadas? (ex: cabeçalho, conteúdo, rodapé, destaques etc.)

Cada bloco será uma div, section, header, main, etc.

Exemplo:

  • 🔹 Topo (logo + menu)
  • 🔹 Linha com 3 cards grandes
  • 🔹 Linha com banner e sugestões menores

🔀 Grupos de elementos em linha ou coluna

Verifique se os elementos estão lado a lado ou empilhados:

  • Se lado a lado → flex-direction: row
  • Se um embaixo do outro → flex-direction: column

Exemplo:

  • Três vídeos: em linha (row)
  • Banner + sugestões: linha com colunas dentro (com listas em coluna)

📱 Elementos responsivos

Pense: “O que deve mudar em telas pequenas?”

  • Cards que estavam em linha devem se empilhar?
  • Uma coluna de lado deve descer para baixo do conteúdo?

Isso afeta como você configura o flex-wrap e os @media queries.

✅ Etapa 2: Estrutura HTML Base

Com o layout compreendido, escreva a estrutura HTML pura, sem CSS.

⛓️ Dica: pense em "caixas dentro de caixas".

<body>
  <header>...</header>
  <main>
    <section class="linha"> <!-- cards grandes --></section>
    <section class="listas"> <!-- banner + listas --></section>
  </main>
</body>

✅ Etapa 3: Configuração do Flexbox

Com CSS, use Flexbox nos contêineres principais.

📦 display: flex

.linha {
  display: flex;
  flex-direction: row; /* ou column */
  gap: 20px;
}

🎯 Distribuição e alinhamento

  • justify-content: distribui no eixo principal (ex: space-between, center)
  • align-items: alinha no eixo cruzado (topo, centro, base etc.)

✅ Etapa 4: Tamanho e Comportamento

💡 Use flex: 1 1 300px;

  • 1 = pode crescer
  • 1 = pode encolher
  • 300px = tamanho base

Isso permite que os cards se ajustem automaticamente ao espaço.

📏 Use max-width ou min-width:

.card {
  flex: 1 1 300px;
  max-width: 400px;
}

✅ Etapa 5: Imagens e Textos

🖼️ object-fit: cover

Faz com que a imagem preencha a caixa proporcionalmente, cortando as sobras sem distorcer.

.card img {
  width: 100%;
  height: 180px;
  object-fit: cover;
}

🎯 Centralizar conteúdo

.card {
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
}

📌 Resumo Visual

Etapa Ação Objetivo
1. Análise Observar blocos, linhas e colunas Entender estrutura geral
2. HTML Base Criar contêineres para os grupos visuais Organizar o conteúdo em caixas
3. Flexbox display: flex, gap, justify-content Distribuir filhos no layout
4. Tamanho flex: 1 1 base, max-width Permitir responsividade, evitar deformações
5. Imagem object-fit: cover Imagem bonita sem distorção
5. Centralizar justify-content + align-items Centralizar texto, imagem ou ícones
